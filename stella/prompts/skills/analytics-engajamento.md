# Analytics de Engajamento

> ⚠️ **Pré-requisito:** Esta skill é aspiracional. Requer dados reais do Instagram Insights (Sub-projeto C4 — Analytics ativo). Sem dados, use os frameworks como heurísticas de planejamento.

## Métricas-chave por objetivo

| Objetivo | Métrica primária | Métrica secundária |
|---|---|---|
| Alcance de topo | Impressões únicas | Compartilhamentos |
| Engajamento ativo | Taxa de comentários | Saves |
| Conversão | Cliques no link (bio) | DMs iniciados |
| Retenção | Taxa de conclusão Reels | Novos seguidores/post |

## Benchmarks Instagram (conta educação/IA, 1k-10k seguidores)

- Taxa de engajamento (likes+comentários / alcance): ≥ 3% = bom, ≥ 6% = excelente
- Save rate (saves / impressões): ≥ 1.5% = conteúdo de valor
- Share rate (compartilhamentos / impressões): ≥ 0.5% = viral potential

## Framework de diagnóstico de post

```
Post com baixo engajamento? →
  Impressões altas, engajamento baixo? → Problema no hook (não parou o scroll)
  Impressões baixas? → Problema no horário OU shadow penalidade
  Saves altos, comentários baixos? → Conteúdo educativo (ok para awareness)
  Comentários altos, saves baixos? → Conteúdo opinativo (ok para comunidade)
```

## Uso no planejamento (sem dados reais)

- Pilar 1-2 (topo): otimizar para compartilhamentos (hook emocional/curioso)
- Pilar 3 (nicho): otimizar para saves (conteúdo técnico valioso)
- Pilar 4 (prova): otimizar para DMs (CTA direto para conversa)
