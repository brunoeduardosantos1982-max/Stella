# Revisão de padrões de marca

## Objetivo

Aplicar **critérios objetivos** de qualidade a cada conteúdo antes de publicar. Veredicto: `aprovado` | `refazer` (com motivo específico, não vago).

## Checklist de revisão

### Dimensão: Copy + engajamento

- [ ] Hook (1ª linha) para o scroll? (Não é genérico/"Oi pessoal")
- [ ] CTA é único e claro? (Não pede 3 coisas)
- [ ] Zero jargão sem tradução?
- [ ] Estrutura PAS ou AIDA aplicada?
- [ ] Exemplos/dados/prints concretos (não teoria)?

### Dimensão: Design + visual

- [ ] Hierarquia clara (1 palavra-chave destacada)?
- [ ] Contraste OK em fundo escuro (peso ≥600, cor vibrante)?
- [ ] Max ~20 palavras por slide/post?
- [ ] Tipografia consistente (mesmo font, 1-2 cores principais)?
- [ ] Visual 🔥 em pontos de destaque (capa, payoff, CTA)?

### Dimensão: Marca + posicionamento

- [ ] Tom de voz mktmagneto (confiante, técnico, sem guru)?
- [ ] Público-alvo claro (pra quem é esse post)?
- [ ] Alinha com pilar editorial (topo amplo / nicho / prova social)?
- [ ] Não contradiz posts anteriores (consistência)?

### Dimensão: Meta + performance

- [ ] Hashtags adequadas (12-15, mix correto de amplo/nicho)?
- [ ] Legenda + visual estão sincronizados (não discord)?
- [ ] CTA é acionável em 5 segundos (leitor entende ação)?

## Fraquezas comuns (motivos de refazer)

### ❌ Hook genérico
- *"Oi pessoal, hoje vou falar de IA"* ← nenhuma curiosidade
- **Motivo de refazer:** "Hook não para scroll. Precisa de uma promessa, número, ou pergunta provocadora."
- **Ação:** reescrever 1ª linha

### ❌ CTA difuso
- *"Curte, salva, compartilha com 3 amigos e comenta sua opinião"* ← pede 4 coisas
- **Motivo de refazer:** "CTA é múltiplo. Escolha UM: comentar / salvar / compartilhar."
- **Ação:** remover até 1 CTA claro

### ❌ Jargão sem tradução
- *"Este agente usa function calling em padrão agentic"*
- **Motivo de refazer:** "Jargão sem contexto. Qual é a implicação em português?"
- **Ação:** traduzir / explicar em 1-2 linhas

### ❌ Hype + promessa irreal
- *"Isso vai mudar TUDO no marketing"*
- *"Ganhe R$10k em 1 semana com esse framework"*
- **Motivo de refazer:** "Hype/promessa irreal. Mostre o dado/case real ou remove."
- **Ação:** trocar por afirmação verificável

### ❌ Falta de exemplo concreto
- *"Prompts melhorados aumentam qualidade de resposta"* ← óbvio, ninguém salva isso
- **Motivo de refazer:** "Falta concretude. Adicione: prompt antes/depois, print, ou número."
- **Ação:** buscar exemplo real ou case

## Veredictos

### ✅ APROVADO
- Passa em 90%+ do checklist
- Nenhuma fraqueza crítica (hype, promessa falsa, falta de ação)
- Publicar já

### ⚠️ REFAZER
- Falha em 2+ críterios core (hook, CTA, jargão)
- 1 motivo objetivo específico
- Exemplo: "Hook não para scroll + CTA múltiplo. Refazer 1ª linha e deixar só 'Comenta IA'."

### 🟡 REFAZER_1X (aceitar com aviso)
- Falha em 1 critério menor (ex: hashtags desalinhadas)
- Potencial alto do conteúdo (ângulo único, dado novo)
- **Ação:** refazer 1x, espaço curto (2h-4h). Se volta com mesmo problema, rejeita.

## O "refazer 1x" — quando aceitar vs. rejeitar

**Aceite com aviso quando:**
- Conteúdo é novo/inédito (vale o rework)
- Falha é em dimensão menor (design, hashtag)
- Autor mostra histórico de refinamento

**Rejeite sem aceitar com aviso quando:**
- Hook é genérico (dimension crítica)
- CTA é vago (dimension crítica)
- Já foi "refazer 1x" 2x esse mês (sinal de falta de atenção)

## Fluxo rápido de revisão (5 min por post)

```
1. Ler hook → para scroll? SIM → avançar | NÃO → refazer hook
2. Ler CTA → é único? SIM → avançar | NÃO → refazer CTA
3. Varrer jargão → tem palavra sem tradução? NÃO → avançar | SIM → refazer
4. Olhar visual → hierarquia clara? SIM → avançar | NÃO → refazer design
5. Verificar pilar → alinha editorial? SIM → APROVADO | NÃO → refazer angle
```

Se passa em 5 checks, é verde.

## Output da revisão

Sempre comunique:

```
POST: [titulo]
VEREDICTO: APROVADO | REFAZER | REFAZER_1X

SE REFAZER:
Motivos:
1. [motivo específico] → [ação concreta]
2. [motivo específico] → [ação concreta]

Prazo: [2h | 4h | 1 dia]
```

Nunca: *"Não achei bom"*. Sempre: *"Hook não para scroll (falta promessa). Sugestão: começar com 'Prompts genéricos deixam 40% das respostas abaixo do potencial'."*
