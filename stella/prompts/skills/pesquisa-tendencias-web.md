# Pesquisa de tendências web

## Objetivo

Descobrir **pautas emergentes** (tendências de últimas 2-4 semanas) que ainda não foram saturadas, permitindo criar conteúdo fresh e alto-engajamento no timing certo.

## Fluxo operacional

### 1. Derivar queries a partir de pilares

Cada pilar de conteúdo gera 2-3 queries de pesquisa:

**Pilar: Copywriting + marketing PT-BR**
- Query 1: "tendência copywriting 2026"
- Query 2: "marketing digital brasil maio 2026"
- Query 3: "estratégia redes sociais novo algoritmo"

**Pilar: IA + agentes**
- Query 1: "agentes IA maio 2026"
- Query 2: "function calling llms tendência"
- Query 3: "agentic workflow novo padrão"

**Pilar: Estratégia digital**
- Query 1: "algoritmo instagram 2026 mudança"
- Query 2: "reels vs carrossel melhor engajamento"
- Query 3: "creator economy trends"

### 2. Filtros na busca

Use Brave Search ou Perplexity com:
- **Timeline:** últimos 30 dias (ou 14 dias para máxima frescura)
- **Idioma:** PT-BR primário, EN secundário
- **Tipo:** evite resultados puramente comerciais/anúncios
  - Priorizar: artigos, blogs, newsletters, posts (de gente confiável)
  - Descartar: landing pages de venda, clickbait puro

### 3. Síntese de resultados

Para cada pauta encontrada:
- **1 frase** resumindo a tendência (máx 20 palavras)
- **1-2 implications** (por que importa para mktmagneto/publico)
- **Ângulo único** — qual a perspectiva que ainda não foi coberta?

**Exemplo:**

```
Tendência: "Agentes IA com memory integrada estão substituindo chatbots"
Implicação: Clientes querem assistentes que lembram contexto, não conversas zeradas
Ângulo único: "Como construir agente que não esquece do cliente (e aumenta conversão)"
```

## Como reconhecer pauta "queimada"

Uma pauta está saturada quando:

1. **>50 posts grandes** (criadores com >10k seguidores) nos últimos 14 dias
2. **Mesmos ângulos repetidos** — "7 formas de usar IA", "5 erros em prompts" (mesmo framework, diferentes autores)
3. **Busca de trending topic** — #ia, #agentes, #copywriting aparecem em múltiplas listas semanais
4. **Dificuldade de diferenciar** — você não consegue pensar em um ângulo único que não foi coberto

**Ação:** descartar ou esperar 2-3 semanas para retomar com ângulo muito diferente.

## Pauta "pronta" vs pauta "ainda quente"

- **Pronta:** <20 posts grandes, ângulos começam a repetir, você tem ângulo único claro
- **Ainda quente:** <10 posts grandes, múltiplas perspectivas possíveis, timing é ouro

Priorizar pauta "ainda quente".

## Evitando redundância entre pautas

Após sintetizar 3-5 pautas potenciais:

1. Compare os "ângulos únicos"
2. Se dois ângulos são muito parecidos, escolha o com mais evidence (dados, case, expert quote)
3. Espaçar publicação: não duas pautas do mesmo subtema em <1 semana

## Ferramentas recomendadas

- **Brave Search:** filtro de timeline, resultado menos clickbait, integra X/LinkedIn
- **Perplexity (Claude mode):** sintetiza múltiplas fontes, connections inteligentes
- **Google Trends:** para validar volume de busca (tendência real vs hype passageiro)

## Log de pesquisa

Mantenha um spreadsheet/doc:

```
| Data       | Query          | Pauta               | Status      | Publicado? |
|------------|----------------|---------------------|-------------|-----------|
| 2026-05-19 | "agentes IA"   | Memory integrada    | Pronta      | Sim       |
| 2026-05-19 | "copywriting"  | Hooks emergentes    | Saturada    | Não       |
| 2026-05-20 | "reel tendencia"| Shorts vs reels     | Ainda quente| Agendado  |
```

Isso previne trabalho duplicado e identifica gaps.
